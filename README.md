## 🤝 如何参与

1. **提 Issue**：反馈问题、讨论新主题  
2. **Fork & PR**：在对应目录中新增或完善 `.md` 文档  
3. **Issue 模板**：请简要说明 “年级 + GitHub ID + 学期 + 课程/项目 + 建议/心得”  
4. **保持更新**：定期同步课程官网信息
